# ML-tools

这是由 [Midairlogn](https://github.com/midairlogn) 创作或参与制作的一些小工具的集合。

> **注意**：这些小工具仅作为学习交流用途，使用时请保证遵守当地的法律法规，请勿用于非法用途。

## 各工具简介

### [Stimulate_Paste](/Stimulate_paste/)

一个 Windows 控制台工具，依赖 Windows 输入模拟机制实现自由粘贴。

### [中国大学慕课AI答题助手](https://github.com/Camvanaa/examFucker/tree/main/MOOC) ([Camvanaa](https://github.com/Camvanaa))

一个基于 Tampermonkey 的油猴脚本，用于自动提取中国大学慕课 ( icourse163.org ) 页面的题目信息并调用AI接口辅助作答。

### [ChatBomb](/ChatBomb/)

`QQ/TIM/微信/Discord` 等聊天软件的刷屏工具。

### [ChatBomb-florr](/ChatBomb-florr/)

`florr` 专用的快捷聊天工具，可以方便你发`ygg`,`tks`等。
